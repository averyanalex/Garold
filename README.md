# Garold
Открой файл example_config.yml, скопируй его содержимое.
Создай новый файл config.yml, вставь в него то, что скопировал.
Перейди на https://discord.com/developers/applications, нажми New Application.
Слева раздел Bot, нажми Build a bot. Там будет токен, нажми copy,
вставь его в конфиг вместо ваш_токен. Потом напиши Аверьянову, он скинет тебе
данные для базы данных. Запиши их в config.yml. Запусти бота!
